Ahman lämpömittaritesti

Ajat x-akselilla ovat UTC-aikaa, lisää niihin kesäaikana 3 tuntia, talviaikana 2h. Kuvaaja päivittyy automaattisesti. Tai ainakin pitäisi.

Lämpömittarina langaton 433MHz lämpömittari, jonka signaali vastaanotetaan ohjelmistoradiolla (SDR) käyttäen rtl_433-sovellusta. Tämän palvelun lähdekoodit saa komennolla git clone http://ahma.koodilehto.fi/.git. Minut tavoittaa mm. Twitteristä.

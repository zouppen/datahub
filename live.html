Ahma nyt

Ladataan...

Ajat x-akselilla ovat UTC-aikaa, lisää niihin kesäaikana 3 tuntia, talviaikana 2h. Kuvaaja päivittyy automaattisesti. Tai ainakin pitäisi.

Lämpömittarina on "joka kodin" langaton 433MHz.n lämpömittari, jonka signaali vastaanotetaan ohjelmistoradion (SDR) sovelluksella rtl_433. Tämän palvelun lähdekoodit saa komennolla git clone http://ahma.koodilehto.fi/.git. Minut tavoittaa mm. Twitteristä.
